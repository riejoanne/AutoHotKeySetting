# AutoHotKeySetting
## 針對Fc660M的一些快速鍵(Fc660m.ahk)
1. 新增比較常用到的媒體鍵 
- Ctrl + Alt + Space : Play/Pause
- Ctrl + Alt + Right : Next
- Ctrl + Alt + Left : Previous
- Ctrl + Alt + , : Volume down
- Ctrl + Alt + . : Volume up
- Ctrl + Alt + m : Mute

2. 當ScrollLock啟用時，將部份鍵當作NumberPad(配置參考Asus Swift 514)
- j : 1
- k : 2
- l : 3
- u : 4
- i : 5
- o : 6
- m : 0
- / : +
- ; : -
- p : *
- 0 : /

3. 因為Fn鍵有點遠，要按出F列對我來說有點困難，增加右邊的Alt鍵+上排數字，對應到F1~F12
